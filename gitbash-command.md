# Before Learning git you should know this basic commands

## Git Bash / Linux Terminal Commands

> Note: These are **not the same as Windows PowerShell commands**

## Navigation

```bash
cd ~
# go to home directory
```

```bash
cd folder_name/path
# go to a specific folder
```

```bash
cd ..
# go back to previous directory
```

## Create folders

```bash
mkdir folder_name
# create a folder
```

```bash
mkdir folder1 folder2
# create multiple folders
```

## Create files

```bash
touch file_name
# create a file
```

```bash
touch file1 file2
# create multiple files
```

## List files and folders

```bash
ls
# show files and folders
```

```bash
ls -la
# show all files including hidden
```

## Show current directory

```bash
pwd
# print present working directory
```

## Clear terminal

```bash
clear
# clear screen
```

## Exit terminal

```bash
exit
# close terminal
```

## Keyboard shortcuts

```text
Ctrl + S  -> freeze terminal
Ctrl + Q  -> unfreeze terminal
```

## Rename or move

```bash
mv old_name new_name
# rename or move file/folder
```

```bash
mv -i old_name new_name
# safer rename (asks before overwrite)
```

## Delete

```bash
rm filename
# delete file
```

```bash
rm -r folder_name
# delete folder and everything inside it
```
